神经网络

一，基础神经网络

神经网络就是一个能够自动学习 输入—> 输出 的函数

最简单的神经元（Neuron）：

最基础的神经网络，只有一个神经元。

  • 假设有三个输入：x1 = 身高，x2 = 年龄，x3 = 体重

  • 神经元会给每一个输入都乘一个权重：x1*w1，x2*w2，x3*w3

  • 再全部相加：z = x1*w1 + x2*w2 + x3*w3

  • 再加一个偏置（Bias）： z = x1*w1 + x2*w2 + x3*w3 + b

1.1 激活函数

激活函数是一个非线性函数。若没有激活函数，那么神经网络就是一个直线。

举个例子：

  • 假如要判断年龄 学历 收入 然后是否批准贷款。
  • 但真实情况不是年龄越大，贷款越容易。
  • 可能是 收入高 学历高 年龄适中 ，更容易获批贷款。

这种关系很复杂，没有激活函数，神经网络只能走直线，有了激活函数，才能走曲线，才能拟合现实世界。

二，RNN循环神经网络

RNN循环网络的特点：

  • 在读取一句话的时候，读取第一个字，进行理解，然后读取第二个字，把第一个字的理解传过来，读取第三个字的时候，把第一个和第二个字的理解传递过来，依次类推。（类似一个流水线）

弊端：

  • 易丢失前面的信息。（流水线越长，前面的信息越容易被稀释）
  • 不能并行，必须排队。（必须按顺序，第二个字要等待第一个字理解完）
  • 训练起来巨慢，长距离信息丢光。

三，CNN卷积神经网络

传统的神经网络在处理图片问题时，会遇见两个问题。

  • 由于像素点数量众多，计算量会非常大，对于每一个像素点，传统的神经网络都需要和其他的像素点建立起连接。
  • 而且会忽略图像的空间结构信息。例如我们识别下面猫的基本特征时，要先识别猫的耳朵（像素点01），猫的脚趾（像素点31）。
  • 对于神经网络来说，也应该用这种方式去理解猫的这个物种，但是对应到像素块，进行强行学习，费时费力也容易学习错误。

理想状态：神经网络在识别图像时，可以专注局部区域，进行更精确的特征提取。例如在识别猫耳朵的时候，只需要关注猫耳朵的像素点，从而总结出耳朵的特征描述。

3.1 卷积核

假设设定一个2*2范围的像素块作为卷积核，并设置一个随机的k值，来让ai自主学习。每当卷积个经过一个像素块，就让K1*00+k2*01+k3*10+k4*11得出一个新值，这个值就是这个像素块集合的特征。

但是图像是由RGB（红绿蓝）三色叠加而来的。所以不仅局限于简单的二维平面，而是需要在每个颜色层面进行特征提取。

不同颜色层，同一像素位置，是同一像素特征，所以也要采用相同的方式处理

但是实际情况中，一个卷积核只能得出一个特征，所以要构建多份卷积核，每份卷积核都以不同的参数初始化，从而捕捉图像的不同特征。
