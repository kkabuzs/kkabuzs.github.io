GPU CUDA 概述

一，CUDA是什么

NVIDIA 提供的一整套 GPU 并行计算平台（Parallel Computing Platform）和编程模型（Programming Model）。

包括：

CUDA作用：
如何让 CPU 控制 GPU 工作。

解释：
CPU 和 GPU 通常是相对独立的设备。GPU 一般不会主动运行主程序，而是由 CPU 端程序通过 CUDA、图形 API 或其他 GPU 运行时/驱动接口提交任务，随后 GPU 执行这些任务。

二，CUDA 工作流程

CUDA 的工作流程其实只有四步：

CUDA中的两个重要角色：

  • Host：CPU
  • Device：GPU
  • Host Memory 和 Device Memory 默认不共享。

三，CUDA 编程模型（Programming Model）

介绍：
CUDA Programming Model 就是如何把一个任务拆成几万个线程去执行。

四，Kernel（核函数）

介绍：
CUDA 新增加了一种函数，就是核函数。

与普通函数的区别：

  • 普通函数：CPU 调 CPU
  • Kernel：CPU 调 GPU

4.2 Thread（线程）

Thread（线程）是 CUDA 中最小的逻辑执行单位。

启动一个 Kernel 后，大量线程执行相同代码，但通常通过各自的编号处理不同数据：
