二，构建一个基础神经网络

说明：

  • torch：PyTorch的核心功能（核心模块）。
    • 作用：提供pytorch基础能力。
      1. Tensor（张量）
      2. 数学运算
      3. GPU（CUDA）支持
      4. 随机数生成
      5. 自动求导（Autograd）
  • touch.nn：构建神经网络组件
    • 作用：提供构建神经网络所需的工具
      1. nn.Linear（全连接层）
      2. nn.Conv2d（卷积层）
      3. nn.ReLU（激活函数）
      4. nn.GELU（激活函数）
      5. nn.CrossEntropyLoss（损失函数）
      6. nn.Module（基类）
  • torch.optim：训练神经网络的工具
    • 作用：训练神经网络时，需要不断更新模型参数（权重）。常见的优化器如下：
      1. optim.SGD：随机梯度下降。 （在模型训练过程中，根据损失函数计算出的梯度，更新模型参数，让损失值逐渐减小。）
      2. optim.Adam：自适应矩估计。 （使每个参数都有自己的“学习率”，并且这个学习率会根据该参数的历史梯度大小自动调整。）
      3. optim.AdamW：Adam 的修正版本。 （修复了 权重衰减（weight decay） 的 Adam，泛化能力更好。）

2.1 创建数据

解释：

  • distances：配送距离的张量（Tensor）。共四条数据，每条数据一个特征。dtype表示为存储32位浮点数。
  • times：配送时间张量。

2.2 模型构建

解释：

  • nn.Sequential：是一个 容器模块（container module），用于将多个神经网络层按顺序堆叠起来，形成一个复合模型。它的作用类似于一个“顺序执行的模块集合”。
  • nn.Linear(1, 1)：是一个全连接层（也叫做线性层），它有一个输入特征和一个输出特征。
  • 当调用 model(input)，会自动应用 nn.Linear(1,1) 层，输入会被送到该层进行处理。

这个神经元将自动学习最佳的权重和最佳的偏差。

定义损失函数

解释：

  • nn.MSELoss()：均方误差损失函数（Mean Squared Error Loss）。 是 PyTorch 内置的回归任务常用损失函数之一，计算预测值与真实值的差值平方，再按配置做聚合（默认取平均值），数值越小代表预测结果越贴近真实值。
  • optim.SGD(model.parameters(), lr=0.01)：随机梯度下降优化器（Stochastic Gradient Descent）。 调整权重和偏差的方向以调整误差。
  • model.parameters()：核心作用是收集并返回模型中所有可训练的参数（权重、偏置张量）。 model.parameters() 会自动遍历模型的所有层，把所有可训练的参数张量汇总成一个迭代器，一次性交给优化器SGD。
  • lr=0.01：learning rate（学习率） 的缩写，控制模型参数更新步长。 lr=0.01 表示将 SGD 优化器的学习率设置为 0.01。

2.3 模型训练

说明：

  • optimizer.zero_grad()：清除前一轮训练的所有计算的值。 若不加，PyTorch会在各轮中累积调整，扰乱所有学习。
  • outputs = model(inputs)：使用距离作为输入。
  • loss = loss_function(outputs, times)：将预测值与真实时间进行比较并计算误差。
  • loss.backward()：计算如何调整权重和偏差以调整误差，计算术语叫反向传播。
  • optimizer.step()：将上面所有的计算进行调整。

2.4 模型评估

说明：

  • with torch.no_grad():：梯度禁用上下文管理器。模型推理 / 预测时，只需要前向传播得到输出，不需要反向传播更新参数。关闭梯度可以大幅节省显存、提升运算速度。
  • test_distance = torch.tensor([[[25.0]]], dtype=torch.float32)：构造测试张量。
  • predicted_time = model(test_distance)：向前传播。 将测试数据传入已经训练好的模型，得出预测结果predicted_time，结果输出格式为张量。
